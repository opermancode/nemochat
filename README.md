# NemoChat 🚀

Isolated, request-based chat system built for security and scalability.

## 🛠 Tech Stack
- **Frontend:** Next.js 14, Tailwind CSS
- **Backend:** Node.js, Express, Socket.io
- **DB:** MySQL 8.0
- **Infra:** Docker, Nginx, AWS EC2
